Welcome to the jOpt Project

jOpt is an open source implementation of the Optimization Programming Language (OPL).

JS, the first add-on to the CSP module, has been released. Check the news page for details and the download page to try it out. (Jan 10, 2007)

jOpt has released a new and improved version of its CSP component. Visit the downloads page to obtain the latest and greatest. (Dec 17, 2006)

Development continues on the JS module as well as the Simplex-based MP solver.

For more information, please contact mystafer@users.sourceforge.net or cmj917@users.sourceforge.net.
