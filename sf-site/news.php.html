jOpt Site Changes

Updated Site - Jan 19, 2007
The jOpt website has undergone several updates. Information has been revised and corrected in a variety of sections including the front-page, the FAQs, and the explanations of OPL, MP, and CP. Also, the Documentation and Links sections have been redone, and the JS component has been added.


jOpt Announcements

JS Alpha 1.0 - Jan 10, 2007
JS, the first add-on to the CSP component of jOpt, has been released. JS is focused on the modeling and solving of Job Scheduling problems and runs "on top of" the CSP component (ie. JS requires CSP for running/compilation). JS Alpha 1.0 is available from the sourceforge downloads site:
Downloads Page


JS Javadocs Available - Jan 10, 2007
The Javadocs for the official JS Alpha 1.0 release are available on this site.


CSP Release 1.0 - Dec 17, 2006
A new, more stable, more complete, more efficient version of the CSP portion of the jOpt project is done. CSP Release 1.0 is available from the sourceforge downloads site:
Downloads Page


CSP Javadocs Updated - Dec 18, 2006
The updated Javadocs for the official CSP Release 1.0 are available on this site.
